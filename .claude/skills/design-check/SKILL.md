# Проверка дизайна командой агентов

Запусти **4 параллельных агента** для проверки текущего состояния сайта:

## Агент 1: Дизайнер
Проверяет:
- Выравнивание элементов (flex justify-between для заголовков)
- Отступы, spacing, консистентность
- Цветовая палитра (primary: #1B2A4A, accent: #D4A843)
- Шрифты (Montserrat заголовки, Inter текст)
- Анимации Framer Motion — плавные, не раздражают
- Адаптивность на всех экранах

## Агент 2: Копирайтер
Проверяет:
- Орфография и грамматика (русский язык)
- Профессиональный тон строительной компании
- Описания проектов — информативные, не шаблонные
- CTA кнопки — понятные призывы к действию
- SEO тексты — ключевые слова естественно вписаны

## Агент 3: Кодер
Проверяет:
- Нет ошибок TypeScript
- Компоненты оптимизированы (нет лишних ре-рендеров)
- Изображения с CDN загружаются корректно
- API endpoint `/api/contact` работает
- Нет console.log или дебаг-кода

## Агент 4: Босс (финальная оценка)
Собирает отчёты всех агентов и выдаёт:
- Общая оценка 1-10
- Топ-3 проблемы для исправления
- Что хорошо и не трогать

## Компоненты сайта
Проверяй файлы в `src/components/`: Header, Hero, About, Services, Portfolio, Process, Stats, Contacts, Footer

## Стиль проекта
- Tailwind CSS v4
- CSS переменные в `src/app/globals.css`
- Тёмная тема: фон #0D1117, текст белый/серый
