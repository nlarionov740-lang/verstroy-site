# Добавить новый проект в портфолио

Пользователь хочет добавить новый объект: $ARGUMENTS

## Структура данных

Файл: `src/data/projects.ts`

Каждый проект имеет формат:
```typescript
{
  id: "slug-proekta",           // латиница, через дефис
  name: "Название объекта",     // на русском
  slug: "slug-proekta",        // совпадает с id
  type: "Монолит",             // тип работ
  location: "г. Пермь",        // город
  description: "Описание...",  // 1-2 предложения о работах
  photoCount: 0,               // количество фото (0 если ещё нет)
  coverIndex: 1,               // какое фото использовать как обложку
}
```

## Действия

1. **Прочитай** `src/data/projects.ts` — пойми текущую структуру и категории
2. **Спроси** у пользователя недостающие данные:
   - Название объекта
   - Тип работ (Монолит, Фасад, Кладка, Кровля или другой)
   - Город/локация
   - Краткое описание работ
   - Количество фото (если есть)
   - Номер фото для обложки
3. **Сгенерируй slug** из названия (латиница, lowercase, дефисы)
4. **Добавь** объект в правильную категорию (по комментариям в файле)
5. **Покажи** результат пользователю

## Важно
- Slug должен быть уникальным
- Описание — 1-2 предложения, профессиональный тон
- Добавляй в правильную секцию по типу работ
- Если тип новый — создай новый комментарий-разделитель
