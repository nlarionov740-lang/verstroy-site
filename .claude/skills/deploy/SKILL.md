# Деплой сайта на Vercel

Выполни последовательно:

1. **Проверь статус git** — есть ли незакоммиченные изменения
2. **Запусти сборку** — `npm run build` в корне проекта
3. **Если сборка упала** — покажи ошибки и предложи исправления, НЕ деплой
4. **Если сборка успешна** — закоммить все изменения с осмысленным сообщением
5. **Пуш на GitHub** — `git push origin main`
6. **Подтверди деплой** — Vercel подхватит автоматически

Все долгие команды (build, push) запускай в фоне.

Если на любом этапе возникла ошибка — остановись и сообщи пользователю.
